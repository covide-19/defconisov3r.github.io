<h2><span class="mw-headline" id="The_first_rule:_do_no_harm">The first rule: do no harm</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Covid19Relay&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Edit section: The first rule: do no harm">edit</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Covid19Relay&amp;action=edit&amp;section=1" title="Edit section: The first rule: do no harm">edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Inclusion criteria on this wiki is more flexible compared to Wikipedia and the Wikimedia wikis. This is intentional. However, the primary rule is that no content should be harmful to others. A corollary of this: personally identifiable information, especially about cases, is usually harmful.
</p>
<h2><span class="mw-headline" id="How_it_works">How it works</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Covid19Relay&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Edit section: How it works">edit</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Covid19Relay&amp;action=edit&amp;section=2" title="Edit section: How it works">edit source</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>This is a wiki, like the others. To create a regular text page, go to that page title and create it.
</p><p><b>Wikibase support!</b> This is an experiment. No one is quite sure if this is going to be a sandbox for future <a rel="nofollow" class="external text" href="https://www.wikidata.org">Wikidata</a> entries, a database all on its own, or something in between. To create items and properties, go to <a href="/wiki/Special:NewProperty" title="Special:NewProperty">Special:CreateProperty</a> and <a href="/wiki/Special:NewItem" title="Special:NewItem">Special:CreateItem</a>. There is a <a rel="nofollow" class="external text" href="http://covid19-backend.popup.wiki:8282/">SPARQL query service</a> and <a rel="nofollow" class="external text" href="http://covid19-backend.popup.wiki:9191/#/">QuickStatements app</a>.
</p><p><b>File uploads!</b> Coming soon. No restrictions on copyright licenses or file formats. Do your best to provide attribution. No piracy. We will honor DMCA takedown requests that are valid. If you <i>can</i> upload it to <a rel="nofollow" class="external text" href="https://commons.wikimedia.org">Wikimedia Commons</a>, you <i>should</i> upload it to Wikimedia Commons.
</p>
